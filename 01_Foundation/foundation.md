# Foundation Layer

## Core Topics

- **Linux basics**: Filesystem hierarchy, permissions, process management, shell utilities.
  - *Reference*: [Linux Journey – Essentials](https://linuxjourney.com/lesson/essentials)
  - *Reference*: [The Linux Command Line by William Shotts (Free PDF)](https://github.com/astley/devcli/blob/master/books/The%20Linux%20Command%20Line.pdf)
- **Networking fundamentals**: OSI model, TCP/UDP, IP addressing, subnetting, routing.
  - *Reference*: [Cisco Networking Basics](https://www.cisco.com/c/en/us/solutions/enterprise-networks/networking-basics.html)
  - *Reference*: [Subnetting Made Easy (YouTube)](https://www.youtube.com/watch?v=5hMqnVUd3wI)
- **HTTP, DNS, TLS**: Request/response cycle, status codes, DNS resolution, TLS handshake, certificates.
  - *Reference*: [MDN Web Docs – HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP)
  - *Reference*: [Let’s Encrypt – How TLS works](https://letsencrypt.org/how-it-works/)
- **Git and GitHub**: Commit workflow, branching, merging, rebasing, pull requests, CI integration.
  - *Reference*: [Pro Git (free book)](https://git-scm.com/book/en/v2)
  - *Reference*: [GitHub Guides – Understanding the GitHub Flow](https://guides.github.com/introduction/flow/)
- **Bash or Python scripting**: Automation, text processing, REST API interaction.
  - *Reference*: [Automate the Boring Stuff with Python (Free)](https://automatetheboringstuff.com/)
  - *Reference*: [Bash Guide for Beginners](http://tldp.org/LDP/Bash-Beginners-Guide/html/)
- **Basic cloud concepts**: IaaS vs PaaS vs SaaS, regions, availability zones, resource provisioning.
  - *Reference*: [AWS Cloud Practitioner Essentials (PDF)](https://d1.awsstatic.com/training-and-certification/docs-cloud-practitioner/AWS_Cloud_Practitioner_Essentials.pdf)
- **Debugging and communication**: Log inspection, error tracing, effective written and verbal reporting.
  - *Reference*: [Google SRE Book – Incident Response](https://sre.google/sre-book/handling-incident/)

These fundamentals form the foundation required to understand and secure more advanced systems.
